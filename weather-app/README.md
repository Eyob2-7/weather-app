Description
A React app that fetches and displays hourly temperature data using the Open-Meteo API. Enter latitude and longitude to get real-time weather updates for any location.

Features
Fetch weather data for up to 3 days.
Responsive design with a weather-themed background.
Displays hourly temperatures in a user-friendly format.
How to Run
Clone the repository:
git clone https://github.com/your-username/weather-app.git

cd weather-app
Install dependencies:
npm install

Add a .env file with the following two properties:
REACT_APP_DEFAULT_LATITUDE=52.52
REACT_APP_DEFAULT_LONGITUDE=13.41

Start the app:
npm start

API Used
Open-Meteo API
Fetches hourly temperature data using latitude and longitude.
